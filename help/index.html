Справка по приложению FallSimulation

FallSimulation - редактор уровней для этой же игры. Выполнена в стиле Марио.
OpenSource

Разработчик: Чечкенёв Андрей Дмитриевич/DarkCat09
Ссылка на Github

После запуска приложения Вы увидите главное меню.
В нём есть кнопки:

Start для запуска редактора;

Exit для выхода из приложения
(выход также может быть произведён нажатием клавиши Q - Quit);

? Help для запуска данной справки;

и Play для проверки готового уровня.
